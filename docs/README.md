# Documentation

This directory contains additional documentation and resources for GFACS.

## Contents

- **orchestrator.md**: Detailed documentation for the experiment orchestrator
- **AGENTS.md**: API documentation for the docs module

## Documentation Structure

GFACS documentation is organized hierarchically:

- **Root README.md**: Main user guide and quick start
- **AGENTS.md files**: Technical API documentation for each module
- **Module READMEs**: Problem-specific usage guides
- **This directory**: Advanced usage and architecture details

## Building Documentation

Documentation is maintained in Markdown format and rendered through GitHub.
